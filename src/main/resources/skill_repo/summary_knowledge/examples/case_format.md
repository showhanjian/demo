# 案例格式

## 成功案例格式

```markdown
## 案例: [任务名称]

### 基本信息
- 日期: YYYY-MM-DD
- 用户: user_id
- 技能: skill_id

### 任务描述
用户需求描述...

### 执行过程
1. 步骤1...
2. 步骤2...

### 成功因素
- 因素1
- 因素2

### 经验总结
学到的经验...
```

## 失败案例格式

```markdown
## 案例: [任务名称] - 失败

### 基本信息
- 日期: YYYY-MM-DD
- 用户: user_id
- 技能: skill_id

### 任务描述
用户需求描述...

### 执行过程
1. 步骤1...
2. 步骤2... (失败)

### 失败原因
失败原因分析...

### 教训总结
教训和避免方法...
```
